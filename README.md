## Build
### Build
```sh
cargo build --release
```
### Build and Run
```sh
cargo run --release
```

## Controls
- With mouse clicks you can draw single cells;
- S or Space - for stop;
- B - for blank screen;
- ESC - for exit;
- R - for randomize;
- +/- - Slowdown/speedup simulation.
